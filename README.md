# SoulHeal - Student Mental Health Support Platform

SoulHeal is a full-stack web application designed to support students in managing their mental health by providing accessible resources and a personalized chatbot experience.

## Features
- User authentication and session management for secure access
- AI-powered chatbot integrated using prompt engineering to provide personalized responses
- Curated mental health resources categorized for student needs
- Emergency contact information for immediate support
- Admin module to manage and update platform content

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT
- AI Integration: Prompt-based chatbot using external API

## How It Works
1. Users sign up and log in securely  
2. Based on user inputs and preferences, the chatbot generates relevant responses  
3. Users can explore categorized resources for guidance and support  
4. Emergency contacts are available for critical situations
5. Users can use chatbot for their mental health issues support or prefer answering a few questions for a personalized experience.
6. Admin can update and manage content directly from the backend  

## Objective
To provide a simple, accessible platform that helps students understand and manage their mental well-being through personalized interaction and curated resources.

## Future Improvements
